# データオンボーディングの流れ

onboarden によるデータオンボーディングの流れを示します。

<img
  src={require("/docs/onboardenの仕組み.png").default}
  alt="onboardenの仕組み"
/>

### 1. テンプレートを作成するを作成

インポートするデータの形を定義します。

詳しくは、[テンプレートを作成する](/getting-started/make-a-template)を参照してください。

### 2. インポーターを設置

あなたのプロダクトにインポーターを設置します。

詳しくは、[インポーターを設置する](/getting-started/embedding#インポーターを設置する-1)を参照してください。

### 3. JWT 生成

インポーターの起動に必要な JWT を生成します。

詳しくは、[JWT を生成する](/getting-started/embedding#jwt-を生成する)を参照してください。

### 4.インポーター経由でファイルをアップロード

インポーターからファイルをアップロードします。

詳しくは [ファイルをアップロードする](/getting-started/test-import#ファイルアップロード)を参照してください。

### 5. JWT 認証 & マッピング・バリデーション

データのマッピングを行います。

詳しくは [マッピング](/getting-started/test-import#マッピング)を参照してください。

### 6-a. 結果データのコールバック

完了したインポートデータをコールバックで受け取ります。

詳しくは [コールバックを受け取る](/getting-started/test-import#コールバックを受け取る)を参照してください。

### 6-c 管理画面からデータをダウンロード

完了したインポートデータを管理画面からダウンロードします。

詳しくは [管理画面からデータをダウンロード](/getting-started/test-import#管理画面からデータをダウンロード)を参照してください。
