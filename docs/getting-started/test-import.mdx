# インポーターを試す

## インポートをやってみる

インポーターの設置をまだしていない場合は、[インポーターを設置する](/getting-started/embedding)を参照してください。

### インポーターを起動

`Import` ボタンを押下すると、インポーターが起動することを確認してください。

![インポーター起動](/docs/インポーター起動.png)

### ファイルアップロード

[テンプレートを作成する](/getting-started/make-a-template) で使用した CSV ファイルをアップロードしてください。

### ヘッダー選択

ここでヘッダー行を選択できます。

正しいヘッダー行を選択して、「ヘッダーを決定」ボタンを押下してください。

![ヘッダー選択](/docs/ヘッダー選択.png)

### マッピング

ここでアップロードファイルの列と、テンプレートの項目をマッピングしていきます。

![列をマッピング](/docs/列をマッピング.png)

### プレビュー

マッピング結果のプレビューを確認します。

また、ここでデータを編集することもできます。

![プレビュー](/docs/プレビュー.png)

### 完了

以上でインポートが完了しました。

![インポート終了](/docs/インポート終了.png)
