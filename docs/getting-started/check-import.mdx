# インポート結果を確認する

インポートしたデータを確認する方法はいくつかあります。

## コールバックを受け取る

インポート完了時にをコールバックを受け取ることができます。

[インポーターを設置する](/getting-started/embedding#インポーターを設置する-1)で作成した HTML の下記部分を修正します。

`onCompleted` パラメーターにコールバックを指定してください。

```js
OnboardenImporter.launch({
  templateId: "テンプレートID",
  jwt: "JWT",
  // 追加
  onCompleted: (result) => console.log(result),
});
```

※ CodeSandbox のコードでは既に実装済みです。

## 管理画面からデータをダウンロード

下記の手順でインポート詳細画面にアクセスします。

1. 左のメニューからインポート画面にアクセス
2. インポート一覧から任意の 1 件のインポートをクリック

「最終ファイルをダウンロード」ボタンを押下すると、インポートデータをダウンロードできます。
