# テンプレートを作成する

インポートするデータの形を定義します。

## アカウント作成

onboarden のアカウントをまだ作成していない場合は、まず [アカウントを作成](https://app.onboarden.io/register) します。

アカウントを作成済みの方は [ログイン](https://app.onboarden.io/login) してください。

## テンプレートを作成

テンプレートを作成していきます。
テンプレートを作成するには 2 つの方法があります。

- CSV ファイルから作成
- 手動で作成

CSV データを既にお持ちであれば CSV ファイルから作成するのが最も簡単です。

ここでは、CSV ファイルから作成する方法を紹介します。

「CSV ファイルから作成」を押下してください。

![テンプレート作成_csvから作成](/docs/テンプレート作成_csvから作成.png)

CSV ファイルをアップロードします。

![CSVファイルから作成_アップロード](/docs/CSVファイルから作成_アップロード.png)

下記のファイルをダウンロードして、アップロードしてください。

[社員名簿サンプル.csv](/docs/社員名簿サンプル.csv)

アップロードが完了したら内容を確認して、「作成」を押下します。

![CSVファイルから作成_確認゙](/docs/CSVファイルから作成_確認.png)

テンプレートが作成できました。

![CSVファイルから作成_完了゙](/docs/CSVファイルから作成_完了.png)

## テンプレート項目を編集

項目の定義を編集できます。

ここでは、例として「有効」項目の設定を変更してみます。

「型」を「真偽値」に変更しましょう。

![テンプレート項目編集゙](/docs/テンプレート項目編集.png)

変更が完了しました。

![テンプレート項目編集゙後](/docs/テンプレート項目編集後.png)
